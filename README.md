# KISS Core
========

## What is it?

Kiss Core Microframework on PHP 5.5+

## Installation guide

```bash
git clone git@github.com:dmitrykuzmenkov/kisscore.git
cd ~/kisscore
./make-app myproj
cd ~/myproj
./run > env/log/run.log &

sudo echo '127.0.0.1 myproj.lo' >> /etc/hosts
```

Open in browser http://myproj.lo
